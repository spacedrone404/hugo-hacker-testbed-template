<br>

Locate your style.css

```
NAME-OF-THE-PROJECT\themes\NAME-OF-THEME\assets\css\style.css 
```

Add following code to apply 30px indentation.

```
p {
	text-indent: 30px;
}
```

For no indentation:

```
p {
	text-indent: 0px;
}
```
or just **remove the whole block of p code.**
